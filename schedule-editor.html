未接続

トークンはこの端末のブラウザ内にのみ保存され、GitHub以外には送信されません。ただし共有・公共のパソコンでは「保存する」にチェックを入れないことをおすすめします。

トークンの作り方
  1. GitHubにログイン → 右上のアイコン → Settings
  2. 左メニュー下部の Developer settings
  3. Personal access tokens → Fine-grained tokens → Generate new token
  4. Repository access で対象のリポジトリだけを選択
  5. Permissions → Contents を「Read and write」に設定
  6. Generate token でできた文字列(ghp_...)を上の欄に貼り付け

JSONを書き出す・反映する

手動で反映する場合(設定なしでもOK)

GitHub連携を設定しない場合は、これまで通り「コピー」または「ダウンロード」した内容を、GitHubのWeb画面で schedule.json を開いて貼り付け→Commit changes、で反映できます。
